# Changelog

All notable changes to this project will be documented in this file.

## [2.0.0] - 2026-03-18

### Added
- Telegram RSS 监控支持
- GitHub Discussion 推送功能
- GitHub Release 图片上传功能
- 图片 URL 自动提取（支持 HTML 实体解码）
- 多 Discussion 仓库支持（图片上传仓库 vs 评论仓库分离）

### Changed
- 重构配置文件结构
- 支持 Discussion 评论仓库和图片上传仓库分离

### Features
- ✅ 支持 RSSHub 监控 Telegram 频道
- ✅ 自动提取图片 URL
- ✅ 发评论到 GitHub Discussion（支持 Markdown 图片展示）
- ✅ 支持多个推送平台（Discord、钉钉、飞书、Telegram、GitHub Discussion）
